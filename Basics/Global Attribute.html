class - specifies one or more class name for the element
id - assigns a unique identifier to the element
style - allow inline css styling
title - provide additional information (tooltip on hover)
hidden - hides the element from view
table index - control the tab order of elements
lang - declares the language elemnt of the content
dir - specifies the text direction
access key - specifies keyboard shortcut to access the element
draggble - specifies if the element is draggble
contenteditable - allow the content of the element to be editable
spellcheck - specifies whether to check spelling for the element
data-* - store custom data attributes
translate - indicate wheather the content should be translated
